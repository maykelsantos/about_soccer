ANLISE DE DADOS SOBRE FUTEBOL INTERNACIONAL

Este projeto visa realizar uma anlise abrangente dos dados de futebol internacional de um dataset encontrado no Kaggle (https://www.kaggle.com/datasets/patateriedata/all-international-football-results).

Utilizando dois conjuntos de dados principais, 'all_matches.csv' e 'countries_names.csv', pretendemos explorar padres temporais, avaliar o desempenho das selees nacionais, investigar a influncia de torneios especficos e criar visualizaes para uma representao mais abrangente.

O projeto visa proporcionar uma compreenso aprofundada do panorama do futebol internacional, identificando padres, tendncias e fornecendo insights sobre o desempenho das selees ao longo do tempo e em diferentes contextos de torneios.

As visualizaes geogrficas adicionam uma dimenso espacial, oferecendo uma representao visual das relaes entre pases no cenrio do futebol internacional.
